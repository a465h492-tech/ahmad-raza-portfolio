Technology Stack

Next.js

Front-end UI, routing, authentication, and dashboard interfaces.

Django + DRF

Backend business logic, REST APIs, and authentication layer.

Supabase

Database for users, products, orders, and reviews.

Cloudinary

File storage for product images and media assets.

Payment Gateway

Secure payment processing for marketplace transactions.

Email Service

Automated email and notification delivery system.

Architecture

User → Next.js SPA → Django REST API → Supabase

Front End

Next.js handles UI rendering, client-side routing, user authentication, and dashboard interfaces with server-side rendering capabilities.

Back End

Django + Django REST Framework provides robust business logic, RESTful APIs, user authentication, and admin interface.

Database

Supabase stores users, product listings, orders, reviews, and marketplace transactions with real-time capabilities.

Deployment

Hosted on Vercel with reverse proxy configuration, Cloudinary for media storage, and integrated payment gateway.
